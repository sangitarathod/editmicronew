== Description ==

Smart Product Gallery Slider for Woocommerce plugin will allows you to display a nice touch friendly product slider on product pages. When you have a large number of images to display and prefers the description to be visible with the images then this is the right solution to apply. It'll replaces standard wooCommerce lightbox to an awesome lightbox which you love for sure.

You'll get freedom to try various sizes of the gallery to match with your theme. If necessary you even can easily override the standard styles by writing your own CSS style. In one words, it works great out of the box.

Its very simple but probably the best Woocommerce products gallery plugin avilable in Plugin Directory.

Feature List:

*   Easy to use and Customization.
*   Easily set Slider width and height.
*   Full control on Arrow Navigation.
*	Full control on Thumb Navigation.
*	Custom Lightbox with advance settings.
*   Total control over slider, such as auto play, layout, duration etc.
*   Three type of slider: Thumb Slider, Vertical Thumbs Slider and Dotted Slider.
*	Free Support, Upgrade and many more.

== Installation ==

Installing Smart Product Gallery Slider is very easy.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Product Gallery Menu to configure the plugin

== OPTION ==

The options are pretty straight forward. If you've a custom theme then you can simply wrap the product slider in a new div tag and add your own classe.

Examin the custom classes we used using a tool like firebug or Google Web Developer tools and prepend your wrapper div class to them in the theme style sheet. Then you can add your own styling.
